# MovieDB
A project using the [TMDB](https://developers.themoviedb.org/) api made with Nextjs.

[Live Demo](https://google.com/)

## Tech
- Next.js
- Sass
- i18n

## Features
- SSG
- ISR
- SPA
- Multi language
- Dark and Light Theme


## Project Setup
```sh
npm install
```

## Project Development Start
```sh
npm run dev
```
## Project Build
```sh
npm run build
```
## Project Production Start
```sh
npm run start
```
